----------------------------------------------------------------

			Temonis Ver 0.10.0.1

----------------------------------------------------------------

・オープンベータ版です。不具合や要望は @oruponu までお願いします。

・サウンドファイルは Settings.json に定義してあります。
　設定画面は、まだ実装しておりません。サウンドファイルの名前や
　設定ファイルを直接書き替えてください。

・緊急地震速報のサウンドについては、受信時に必ず鳴動する
FirstReport と、最大予測震度が変化した場合に鳴動する MaxIntChange があります。
・サウンドを再生しない場合は、Sound フォルダを削除もしくはリネームしてください。


●動作環境
　Microsoft Windows 7/8.1/10
　常時インターネットに接続できる環境


●アンインストール方法
　レジストリは使用しておりません。Temonis フォルダを削除してください。
　このバージョンでは、Temonis フォルダ外のファイル操作は行っておりません。
